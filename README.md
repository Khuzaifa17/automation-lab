# Automation Lab (n8n Workflows)

This repository is my practice + portfolio for **n8n automation workflows**.
Each workflow is saved as an exported **workflow.json** with a short README and test examples.

## Structure
- `n8n/01-data-handling/` → JSON basics: Set / Edit Fields / Merge / Keep keys
- More workflows will be added over time.

## How to use
1. Open n8n
2. Import any `workflow.json`
3. Execute and test using the sample URLs in that workflow folder README.

## Notes
- No secrets/tokens are included in this repo.
- Test data is sample/demo only.
